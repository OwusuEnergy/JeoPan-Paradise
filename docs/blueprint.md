# **App Name**: Jeopan Paradise

## Core Features:

- Homepage: Showcase luxurious hostel through hero video, booking widget, highlights, room gallery, testimonials, local experiences, call-to-action, and blog previews.
- Interactive Booking Widget: Enable availability checks and direct bookings via a sticky booking bar that contains check-in/out datepickers, number of guests dropdown, and room type selector.
- Dynamic Content Management: Enable content updates of blog posts, testimonials, and room availability through CMS integration, like Wordpress.
- API integration: API integration to provide all functionalities requested by the client.

## Style Guidelines:

- Primary color: Rich, deep green (#2A4C3F) to evoke a sense of natural luxury and paradise.
- Background color: Soft, desaturated off-white (#F8F8F8) to create a clean, inviting backdrop and support a light color scheme.
- Accent color: Warm terracotta (#C44536) to highlight call-to-actions and draw attention to key interactive elements.
- Headings: 'Playfair', serif; for an elegant, fashionable, high-end feel. Note: currently only Google Fonts are supported.
- Body: 'Inter', sans-serif; providing a modern and neutral text.
- Use custom, elegant line icons that animate on scroll to highlight hostel features.
- Employ masonry-style grids and interactive category filters for galleries to create dynamic layouts.
- Use slow, deliberate animations, like fade-ins, slide-ups, and gentle scale transforms, to enhance user engagement with a duration between 800ms and 1200ms.
